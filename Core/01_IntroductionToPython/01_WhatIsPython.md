### **1.5 How Python Works**

Python is an **interpreted** language ‚Äî meaning:

1. You write your code in a `.py` file.
2. The **Python interpreter** reads the code line by line.
3. It **translates** it into **bytecode**.
4. The **Python Virtual Machine (PVM)** executes that bytecode.

This process makes debugging and testing code much faster.
