### **1.2.1 A Brief History of Python**

**Python** was created by **Guido van Rossum** in the late **1980s** and officially released in **1991**.
Guido worked at **Centrum Wiskunde & Informatica (CWI)** in the Netherlands, where he wanted to design a language that was powerful yet easy to read and use.

Here‚Äôs a timeline of important milestones in Python‚Äôs history:

| **Year**       | **Event**                                                                                                                         |
| -------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| **Late 1980s** | Guido van Rossum starts working on Python as a hobby project.                                                                     |
| **1991**       | **Python 0.9.0** released ‚Äî included functions, exceptions, and core data types like lists, strings, and dicts.                   |
| **1994**       | **Python 1.0** ‚Äî added modules, exceptions, and basic tools.                                                                      |
| **2000**       | **Python 2.0** ‚Äî introduced list comprehensions and garbage collection.                                                           |
| **2008**       | **Python 3.0** released ‚Äî a major upgrade with cleaner syntax and better Unicode support (not backward compatible with Python 2). |
| **2020**       | **Python 2 officially retired** ‚Äî Python 3 became the standard.                                                                   |
| **Today**      | The latest version (Python 3.x) is widely used in AI, data science, web development, and automation.                              |
